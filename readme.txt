python package: NetBasedDSim

the package is designed for disease similarity.

# ----------------------------------------------------------------------------------------
the Common folder is the functions used in many .py files.
the Dataset folder contains several types associations selected from other databases.
the Util folder is some tools for file.
the DisSim folder contains the methods for disease similarity.


the Main.py is the entrance for the package.

# ------------------------------------------------------------------------------------------
NetBasedDSim need several packages:
gensim-3.6.0
networkx-2.1
numpy-1.17.4
fastdtw-0.3.4
python-igraph-0.8.0
